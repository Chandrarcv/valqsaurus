<div style={{textAlign: 'justify'}}>
With the Plan Tab, you can create new plans based on existing data (e.g. prior period actuals or existing budget)
or from scratch (Zero-based budgeting).
<br /> <br />
On clicking the Plan Tab, the 'New Series' popup opens (see image below). Before creating a new series the user is required
to click on the 'Lock' button. This ensures that the Node hierarchy is fixed and no structural changes are possible unless 
unlocked again.
</div> <br />
