Plan Features

Let's take a look at the features of the Plan tab in this section.

#1. View Current Plan

You can see the name of the series you are currently working on as highlighted in the figure below. The footer highlights the base series on which the plan has been created.


By clicking on the settings icon, the newly created series can be set as primary or comparison series for the Plan tab. You can also rename or delete the series here.


#2. Modify Current Plan

You can modify an existing plan on ValQ by way of adding/ removing nodes, changing the configuration of nodes, etc. The following example shows how to add a new node to an existing plan.

The below image shows an existing plan on ValQ.


Once you have created a plan in the Plan tab, the "Lock" and "Unlock" options get added to the Advance Editor under Node on the Model tab. After clicking the lock option in the Plan tab, the model gets locked and now you cannot edit the configuration for the nodes (see figure below).


In case you want to make changes to the model, click the unlock option. You will see a pop-up with a warning message as displayed in the image below. Select yes to continue.


Now make changes to the model and lock it again to view the updated plan. Here, node 'Other' has been added as seen in the below image.


Select Yes to confirm locking the model.


The plan has been updated as the below figure.


#3. Add New Plan

Click on the '+ New Series' to add another plan to the Plan tab.


#4. Search for a Node

You can take a look at a particular node and it's children by entering the node name or ID using the search bar. The model along with period values get displayed in a table form. In case, the empty series has been chosen as the series to start with, only the structure and relationships are displayed and not the values.


#5. View the Node Configuration

On clicking any cell with value, the number format and scaling that were configured while modeling is retained. You can see the absolute value on the 'Value Bar' and the Calculation Method and Time Aggregation on the footer as highlighted in the below image.

Note
Just like a formula bar in spreadsheets, you can perform basic operations on a value in the value bar


#6. Action Menu

In the 'Action Menuâ, you will find various options such as follows


#6.1 Weight

You can edit the original total value - manually or by simply, adding or subtracting to get a new total value. Weight distributes a node's new total value based on another node chosen by the user. Weight has options to choose series and node by the user. With the change action option, the user can choose to Overwrite Values or Append Values.


Total Value: The node values under the Total column

Period Value: The node values under the period (month) columns

Overwrite Values: The original value is overwritten by the new value and this new value gets distributed to the spread on top of the original value

Append Values: The original value is appended by the change (i.e. the difference between the original value and new value) and the change gets distributed to the spread by appending to the original values of the spread

#6.2 Equals


Like Weight, here you can edit the original total value - manually or by simply, adding or subtracting to get new total value. Equal distributes a node's total value equally over the periods. With the change action option, the user can choose to Overwrite Values or Append Values.

#6.3 Trend

Trend enables the user to increase/decrease the original period values - By Percentage or By Value and then apply the change to the remaining periods of that node. With the change action option, the user can choose to Overwrite Values or Append Values.


Note
The Increase Trend option accepts both positive and negative values.

#6.4 Copy-to

Copy-to option takes a node's period value and applies the same value to the remaining periods of that node. The change gets updated to the parent node as well and is highlighted in yellow as seen below.


#6.5 Auto Forecast

Auto Forecast enables the user to apply Time Series Forecasting methods to the plan. Click here to know more about it.


#6.6 Allocate Children

Child nodes are allocated either by 'Weight' or 'Equal' under this option and 'Copy-to' is used to copy a node's value to its sibling nodes.


#6.7 Hold and Release

Hold and Release can be used to lock or unlock cells as needed. The feature can be applied on cell, period, row or column level. This option helps you to configure the rest of the plan tab and then release the cells kept on hold.


#6.8 Add a comment

The Note helps the user to add, modify or delete comments on the cell level.


#6.9 Refresh Datasource

On clicking Refresh Datasource, the newly created series gets updated if there are any changes to the input data.


#6.10 View Allocation History

View Allocation History is used to view the changes performed and also roll back changes when needed. 'Reset all' would delete all the allocation adjustments made to the series. 'Restore to this step' traces back to the selected step.

#6.11 Show hidden nodes

You have an option to display the nodes that were hidden while building the model under the Model tab. Here, East is the hidden node and on selecting the Show hidden node option, it gets displayed. However, notice that the Overall Result doesn't get affected.


Note
Upon enabling show hidden node, the node stays hidden on the other tabs.
