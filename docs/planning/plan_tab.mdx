<div style={{textAlign: 'justify'}}>
The Plan tab helps the user to create new plans based on existing data (e.g. prior period actuals or existing budget)
or from scratch (Zero-based budgeting).
<br /> <br />
On clicking the Plan tab, you will see the 'New Series' popup open as the image below. Before creating a new series the user
is required to click on the 'Lock' button. This ensures that the structure of the ValQ model is fixed and no changes are possible
unless unlocked again.
</div> <br />
<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/plan/plan1.png')} />
  </Zoom>
</div> <br />
<div style={{textAlign: 'justify'}}>

Upon locking the model, you will be able to name the new series and choose the base series to begin planning.
The new series can be created based on 'Budget', 'Forecast' or from scratch by selecting 'Empty Series' option.
</div> <br />
