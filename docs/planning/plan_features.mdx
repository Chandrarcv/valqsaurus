### 6.1 Weight

You can edit the original total value - manually or by simply, adding or subtracting to get a new total value.
Weight distributes a node's new total value based on another node chosen by the user. Weight has options to choose series
and node by the user. With the change action option, the user can choose to Overwrite Values or Append Values. 

</div>
<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/plan/plan7.png')} />
  </Zoom>
</div> <br />

**Total Value:** The node values under the Total column

**Period Value:** The node values under the period (month) columns

**Overwrite Values:** The original value is overwritten by the new value and this new value gets distributed to the spread on top of the original value

**Append Values:** The original value is appended by the change (i.e. the difference between the original value and new value) and the change gets distributed to the spread by appending to the original values of the spread

### 6.2 Equals

<div style={{textAlign: 'justify'}}> <br />
Like Weight, here you can edit the original total value - manually or by simply, adding or subtracting to get new total value.
Equal distributes a node's total value equally over the periods. With the change action option, the user can choose to Overwrite Values or Append Values.
</div>

### 6.3 Trend

Trend enables the user to increase/decrease the original period values - By Percentage or By Value and then apply the change to the remaining periods of that node.
With the change action option, the user can choose to Overwrite Values or Append Values.
