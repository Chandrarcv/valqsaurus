### 1. Plan Table
You can see the name of the plan you are currently working on as highlighted in the figure below. The footer highlights the
series on which the plan has been created. 
</div>

### 2. Add New Series
Click on the '+ New Series' to add another series to the Plan tab.
</div>
<div style={{textAlign: 'center'}}>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/plan/plan18.png')} />
</div> <br />
<div style={{textAlign: 'justify'}}>

### 3. Search for a Node
You can take a look at a particular node and it's children by entering the node name or ID  using the search bar. 
</div>
<div style={{textAlign: 'center'}}>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/plan/plan6.png')} />
</div> <br />
<div style={{textAlign: 'justify'}}>

### 4. Get Insight on a Value
On clicking any cell with value, the number format and scaling that were configured while modelling are retained. 
You can see the absolute value on the 'Number Bar' and the Calculation
Method and Time Aggregation on the footer as highlighted in the below image. 
</div>
<div style={{textAlign: 'center'}}>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/plan/plan17.png')} />
</div>

<div style={{textAlign: 'justify'}}> <br />

### 5. Action Menu

### 5.a) Weight

Weight distributes a node's total value based on the period values of a node choosen by the user.
</div>
<div style={{textAlign: 'center'}}>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/plan/plan7.png')} />
</div>

<div style={{textAlign: 'justify'}}> <br />

### 5.b) Equal

Equal distributes a node's total value equally over the period.
</div>
<div style={{textAlign: 'center'}}>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/plan/plan8.png')} />
</div>

<div style={{textAlign: 'justify'}}> <br />

### 5.c) Trend

Trend applies the change based on a node's period value to the rest of the year.
</div> 
<div style={{textAlign: 'center'}}>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/plan/plan9.png')} />
</div>

<div style={{textAlign: 'justify'}}> <br />

### 5.d) Copy-to

Copy-to is used to copy a node's period value and apply the same to the rest of the year.
</div>
<div style={{textAlign: 'center'}}>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/plan/plan10.png')} />
</div>

<div style={{textAlign: 'justify'}}> <br />

### 5.e) Auto Forecast

Auto Forecast enables the user to apply Time Series Forecasting methods to the plan. Click here to know more about it. 
</div>
<div style={{textAlign: 'center'}}>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/plan/plan11.png')} />
</div>

<div style={{textAlign: 'justify'}}> <br />

### 5.f) Allocate Children

Child nodes are allocated either by 'Weight' or 'Equal' under this 
option and 'Copy-to' is used to copy a node's value to its sibling nodes.
</div>
<div style={{textAlign: 'center'}}>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/plan/plan12.png')} />
</div>

<div style={{textAlign: 'justify'}}> <br />

### 5.g) Hold and Release
Hold and Release can be used to lock or unlock certain cells as needed. 
</div>
<div style={{textAlign: 'center'}}>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/plan/plan13.png')} />
</div>

<div style={{textAlign: 'justify'}}> <br />

### 5.h) Note

Note helps the user to add, modify or delete notes. 
</div>
<div style={{textAlign: 'center'}}>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/plan/plan14.png')} />
</div>

<div style={{textAlign: 'justify'}}> <br />

### 5.i) Refresh Datasource

On clicking Refresh Datasource, the newly created series gets updated if there are any changes to the input data.
</div>
<div style={{textAlign: 'center'}}>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/plan/plan15.png')} />
</div>

<div style={{textAlign: 'justify'}}> <br />

### 5.j) View History

View History is used to view the changes performed and also roll back changes when needed.
'Reset all' would delete all the allocation adjustments made to the series. 'Restore to this
step' traces back to the selected step.
</div>
