A XOR statement returns a logical 'Exclusive Or' of all arguments.

## Syntax
XOR ( logical_test1, [logical_test2], ... )

## Arguments
logical_test1 – The first condition to be evaluated. This is a required argument.

logical_test2, …  – The additional conditions to be evaluated. These are optional arguments.

## Example
Consider an investment scenario of investing in either stocks or property, but not both.

To achieve this, the formula should be IF (XOR (stocks, property), invested, not invested).
