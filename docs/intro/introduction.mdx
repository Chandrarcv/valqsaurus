## What is ValQ?

ValQ helps businesses run simulations, plans & perform analysis in a very visual and intuitive manner.

<img alt="ValQ logo" src={useBaseUrl('/doc-images/intro/1.png')} />

## What can I use ValQ for?

ValQ supports use cases across functions and is designed to help decision makers and executives simulate 
various scenarios and outcomes to arrive at actionable decisions. ValQ is a powerful, multi-purpose tool 
that can be used for several business cases - including but not limited to: Business Modeling,
What-if Simulations, Scenario Comparison, Planning, Forecasting etc.

<img alt="ValQ uses" src={useBaseUrl('/doc-images/intro/2.png')}

/>

## Who can use ValQ ?

ValQ is for everyone ranging from C-suite executives to analysts across departments from various 
domains. Note that ValQ is not limited to planning alone. For example, customer service, sales or 
procurement KPIs can be visualized and analyzed.

## What do I need to run ValQ ?

ValQ is an independent solution that does not require any infrastructure, database, service or IT support.
All you need to run ValQ is a desktop or mobile version of Microsoft Power BI. 

Note that the desktop version is free, and you can download it from <a href="https://powerbi.microsoft.com/en-us/downloads/" target="_blank">here</a>.
With ValQ available as a standalone visual for Microsoft Power BI,you can get the app from the app store and
get started in a matter of minutes. Click <Link to={useBaseUrl('docs/intro/import')}> here </Link> to get ValQ.

## What sources can ValQ connect to ?

ValQ can connect to all the databases, platforms and other data sources that is supported by Power BI – including but not limited to:

  - Databases such as SQL Server, SSAS, Oracle, Snowflake
  - Platforms and apps such as Salesforce, Dynamics 365, SAP HANA, and SAP BW
  - Other sources such as Excel, Web, JSON, OData
