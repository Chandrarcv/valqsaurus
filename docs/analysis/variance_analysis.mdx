### 1. Set Base Series and Compared Series

<div style={{textAlign: 'justify'}}>

**Variance Analysis** helps compare the values of Base Series with that of the Compared Series for each of the node in the model and 
calculate variances for the nodes.

You can set the **Base Series** and **Compared Series** from the model that you want to compare here. 
</div>
<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/analyze/analyze3.png')} />
  </Zoom>
</div>

### 2.  Set a Scenario as Base Series

<div style={{textAlign: 'justify'}}>
In the Analyze tab, you can implement the scenario that you have created in the Simulate tab as Base Series and compare it with Compared
Series to perform Variance Analysis. This can be done by simply choosing the scenario that you want from the drop down and clicking on Apply scenario option. <br /> <br />
</div>
<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/analyze/analyze2.png')} />
  </Zoom>
</div>

<div style={{textAlign: 'justify'}}>

### 4. Set Base period and Compared period to calculate variances

<br />
<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/analyze/analyze7.png')} />
  </Zoom>

There are few properties in this section

<div style={{textAlign: 'justify'}}>

**Base Period**: By default, this will show the first period in the ‘Simulation Period’ from the ‘Simulate’ tab. 
This is the period that will show up whenever you would like to analyze variance for the ‘current period’.

**Comparison Period**: By default, you can analyze performance vs. the benchmark for the same period. <br />
E.g., You can change the period to July Budget vs. July Forecast by just clicking on the drop down and choosing Jul.
</div>

### 8. Apply Filter to report

<div style={{textAlign: 'justify'}}>

Use the Filter option to analyze critical influencers.  Two filters are available:

**Filter by Parent**: This allows you to analyze variance for a specific branch within the model. For example, you may be interested in analyzing variance for a particular location or product.

**Filter by Variance**: This allows you to identify top ‘n’ analysis for categories showing the most variances for particular period choosen from the drop down.

> **Note** <br /> You have the option to select whether you want to display only positive variances or only negative variances or both.

</div>
