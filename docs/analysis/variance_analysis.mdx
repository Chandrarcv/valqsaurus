The Analyze tab allows you to run advanced analyses on the data. 
<div style={{textAlign: 'justify'}}> 
Currently, you can perform variance analysis from this tab. 
Expect to see additional items in the future such as sensitivity analysis, attribution analysis and more. </div>

## Variance Analysis

Variance Analysis helps analyze variances against a benchmark for specific periods for all the KPIs in the model.<br />
This pane consists of the following options:

### 1. Find Analyze Tab

On clicking the **Analyze** tab, you will be able to view the Variance Analysis option.

<img alt="ValQ logo" src={useBaseUrl('/doc-images/analyze/analyze1.png')} />

### 2. Base vs Compared Series

<div style={{textAlign: 'justify'}}>

**Variance Analysis** helps compare the values and displays variance between Base Series and Compared Series for the
selected period across all nodes in the model.

Set the **Base Series** and **Compared Series** from the model that you want to compare here. 
</div>
<div style={{textAlign: 'center'}}>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/analyze/analyze3.png')} />
</div>

### 3. Base vs Compared Period

<br />
<div style={{textAlign: 'center'}}>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/analyze/analyze6.1.png')} />
</div>

There are few properties in this section

<div style={{textAlign: 'justify'}}>

**Base Period**: By default, this will show the first period in the ‘Simulation Period’ from the ‘Simulate’ tab. This is the period that will show up whenever you would like to analyze variance for the ‘current period’.

**Comparison Period**: By default, you can analyze performance vs. the benchmark for the same period. E.g., July Forecast vs. July Actuals.

</div>

<div style={{textAlign: 'center'}}>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/analyze/analyze6.2.png')} />
</div>

However, if you would like to compare July Forecast vs. Jan Actuals, select ‘Compare to different period’ 

<div style={{textAlign: 'center'}}>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/analyze/analyze6.3.png')} />
</div>

### 4. Choose Time Period of Analysis

<div style={{textAlign: 'justify'}}>

The Period dropdown allows you to specify periods for which you would like to run the comparison.
</div>
<div style={{textAlign: 'center'}}>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/analyze/analyze4.png')} />
</div>

### 5. Apply Filter

<div style={{textAlign: 'justify'}}>

Use the Filter option to analyze critical influencers.  Two filters are available:

**Filter by Parent**: This allows you to analyze variance for a specific branch within the model. For example, you may be interested in analyzing variance for a particular location or product.

**Filter by Variance**: This allows you to identify top ‘n’ analysis for categories showing the most variance.
</div>
<div style={{textAlign: 'center'}}>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/analyze/analyze5.png')} />
</div>

<div style={{textAlign: 'justify'}}>
