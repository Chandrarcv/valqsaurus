Analyze Tab

The Analyze tab allows you to run advanced analyses on the data.

Currently, you can perform variance analysis from this tab. Expect to see additional items in the future such as sensitivity analysis, attribution analysis and more.

#Variance Analysis

Variance Analysis helps analyze variances against a benchmark for specific periods for all the KPIs in the model.
This pane consists of the following options:

Click on any option to know more
1Find Analyze Tab
2Base vs Compared Series
3Base vs Compared Period
4Choose Time Period of Analysis
5Apply Filter
6Show Absolute & % Variance
7Download as Excel
8Variance Analysis Table

#Find Analyze Tab

On clicking the Analyze tab, you will be able to view the Variance Analysis option.

#Base vs Compared Series

Variance Analysis helps compare the values and displays variance between Base Series and Compared Series for the selected period across all nodes in the model.

Set the Base Series and Compared Series from the model that you want to compare here.

#Base vs Compared Period


There are few properties in this section

Base Period: By default, this will show the first period in the âSimulation Periodâ from the âSimulateâ tab. This is the period that will show up whenever you would like to analyze variance for the âcurrent periodâ.

Comparison Period: By default, you can analyze performance vs. the benchmark for the same period. E.g., July Forecast vs. July Actuals.

However, if you would like to compare July Forecast vs. Jan Actuals, select âCompare to different periodâ

#Choose Time Period of Analysis

The Period dropdown allows you to specify periods for which you would like to run the comparison.

#Apply Filter

Use the Filter option to analyze critical influencers. Two filters are available:

Filter by Parent: This allows you to analyze variance for a specific branch within the model. For example, you may be interested in analyzing variance for a particular location or product.

Filter by Variance: This allows you to identify top ânâ analysis for categories showing the most variance.

#Show Absolute & % Variance

In this section you may choose which variances are important to you for analysis. Selecting these options adds them to the table.

#Download as Excel

Clicking on this button allows you to download the results of variance analysis into a spreadsheet.

#Variance Analysis Table

The table displays the output of the variance analysis based on all the parameters chosen above. The positive variance items are shown in green and the negative ones in red.
